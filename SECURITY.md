# 🛡️ Security Policy

## Supported Versions

We actively support and patch security issues on the latest version of the extension.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability, please do **not** open a public issue. Instead, please report it directly to the maintainers.

### How to Report

Please send an email to the repository owner or contact them privately through their GitHub profile. 

In your report, please include:
1. A description of the vulnerability.
2. Steps to reproduce the issue (and a proof of concept if available).
3. The potential impact of the exploit.

We will acknowledge your report within 48 hours and work with you to resolve the issue promptly.
